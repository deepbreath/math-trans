# Privacy Policy

Math Academy Bilingual Translator translates visible text on Math Academy pages.

## Data processed

- Page text selected for translation.
- Extension settings, including selected translation provider and optional API keys.

## Data storage

Settings are stored in Chrome extension storage. API keys, when provided, are stored locally through Chrome extension storage for the signed-in browser profile.

## Data sharing

When translation is enabled, selected page text is sent to the translation provider selected by the user:

- Google Translate web endpoint
- Google Gemini API
- OpenAI API

The extension does not operate an independent backend server and does not sell user data.

## Permissions

The extension runs on Math Academy pages and requests network access only for the supported translation providers.
